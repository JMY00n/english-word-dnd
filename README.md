## 주요 기능

- 영어 단어 드래그 앤 드롭을 통해 문장을 완성하는 게임
- 문제 데이터는 JSON 파일로 관리하여 문제 교체 및 확장 가능
- 영어 단어 카드를 무작위로 섞어 제공
- Drag & Drop 또는 클릭을 통해 문장 구성 및 수정 가능
- 문장 완성 후 정답 확인 기능
  - 정답: 점수 +1
  - 오답: 점수 -1
- 즉각적인 시각 피드백(테두리 색상 변경)으로 정답 여부 표시
- Next 버튼을 통해 점수 변화 없이 다음 문제로 이동
- 반응형 카드형 UI로 웹 중앙에 앱 형태로 표시
