# English Sentence Builder

## 주요 기능 (Key Features)

### 1. Drag & Drop 게임 플레이
- 뒤죽박죽 섞인 단어 카드(`Word`)를 드래그하여 문장 영역(`Drop Box`)에 배치합니다.
- 단어 사이사이에 카드를 끼워 넣을 수 있는 직관적인 UI를 제공합니다.
- **Check** 버튼으로 정답 여부를 즉시 확인하고 피드백(진동/펄스 효과)을 받습니다.

### 2. 60초 타임어택 (Time Attack)
- **GAME START** 버튼을 누르면 60초 카운트다운이 시작됩니다.
- 제한 시간 내에 정답을 맞히면 **+10점**, 틀리면 **-5점**이 부여됩니다.
- 시간이 종료되거나 **STOP** 버튼을 누르면 최종 점수가 기록됩니다.

### 3. 로컬 랭킹 시스템 (Ranking)
- **LocalStorage**를 활용하여 별도의 DB 서버 없이 회원가입 및 로그인이 가능합니다.
- 개인별 **최고 점수(Best Score)** 가 자동 저장됩니다.
- 전체 유저 중 점수가 높은 **Top 5 랭킹**을 모달 창에서 확인할 수 있습니다.

### 4. 모바일 최적화 UI
- 모바일 환경(400x800 해상도 권장)에 최적화된 세로형 레이아웃입니다.
- **Custom Dropdown** 메뉴와 **Modal** 팝업을 통해 깔끔한 사용자 경험을 제공합니다.

## 참고 사항
- 모든 데이터(회원 정보, 점수)는 브라우저의 **LocalStorage**에 저장됩니다.
- 브라우저 캐시를 지우거나 다른 기기로 접속하면 데이터가 연동되지 않습니다.
